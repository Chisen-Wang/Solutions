# 问题解决方案知识库

这个文件夹用来存放你在生活和学习中遇到的问题，以及后续验证过的解决方案。

核心原则：

- `Markdown` 记录，长期可读、可迁移
- 搜索优先，分类辅助
- 一问题一文件，避免重复记录
- 固定字段，方便自己查，也方便以后让 Codex 协助维护

## 目录结构

- `00_收件箱`
  新问题先放这里，先记下来，再补充整理。
- `10_问题库`
  正式归档区，按一级分类存放。
- `20_索引`
  放总览和高频入口，方便回看。
- `99_模板`
  放统一模板，新增问题时直接复制使用。

## 新增一个问题

1. 先复制 `99_模板/问题模板.md`
2. 按规则命名：`YYYY-MM-DD_一级分类_核心关键词_简述.md`
3. 存到 `00_收件箱`
4. 最少先填写：`title`、`domain`、`status`、`keywords`、`aliases`、`问题描述`

## 文件命名规则

格式：

`YYYY-MM-DD_一级分类_核心关键词_简述.md`

示例：

`2026-04-09_学习_英语听力_听不懂连读怎么办.md`

要求：

- 文件名必须包含“分类 + 核心关键词 + 简述”
- 简述尽量控制在 8 到 20 个字
- 同类问题的新场景优先补充到旧文件，不重复建档
- 同义词不要放文件名，放到 `keywords` 和 `aliases`

## 检索顺序

1. 搜核心关键词
2. 搜口语表达或同义词
3. 按分类目录翻找
4. 查看 `20_索引/高频问题.md`

为了让搜索稳定生效，每个正式问题文件尽量满足：

- 至少 3 个 `keywords`
- 至少 1 个 `aliases`
- 问题解决后补上 `一句话结论`

## 归档规则

- 新问题先放 `00_收件箱`
- 有稳定方案后移动到 `10_问题库/对应分类`
- `status` 只允许：`待解决`、`待验证`、`已解决`
- 高频、常查、特别有用的问题，额外加入 `20_索引/高频问题.md`

## 以后如何让 Codex 帮你

你可以直接说：

- “帮我把这个新问题按模板记到收件箱”
- “帮我查一下以前有没有记过失眠相关问题”
- “帮我把这个问题整理归档到问题库”
- “帮我补充这个问题的关键词、别名和一句话结论”
